# MRVPN Manager Panel

Simple VPN monitoring + control panel (Flask + Socket.IO)

## Features
- Live CPU / RAM / Disk
- Network speed
- Interface traffic (like /proc/net/dev)
- Restart VPN service
- Auto-generated login
- Lightweight UI dashboard

## Install

```bash
sudo bash <(curl -s https://raw.githubusercontent.com/sam-soofy/mrvpn-manager-panel/main/install.sh)
```

## Access it:

http://YOUR_SERVER_IP:5000
